# Aeroclub dei Marsi - Sito Web Ufficiale

Sito web statico per l'Aeroclub dei Marsi.

## Struttura del Sito

Il sito è composto dalle seguenti pagine:

- **index.html** - Homepage con panoramica delle attività
- **chi-siamo.html** - Storia e valori dell'Aeroclub
- **aviosuperficie.html** - Informazioni sull'aviosuperficie LIAH
- **scuola-volo.html** - Dettagli sui corsi di volo e formazione PPL
- **eventi.html** - Eventi e attività dell'Aeroclub
- **area-soci.html** - Area riservata con info su tesseramento, clubhouse e prenotazioni
- **velivoli.html** - Schede tecniche dei velivoli Tecnam P96 e P2002 Sierra
- **contatti.html** - Informazioni di contatto
- **galleria.html** - Galleria fotografica (in allestimento)

## Tecnologie Utilizzate

- HTML5 per la struttura
- CSS3 per lo stile (file `styles.css`)
- Design responsive per dispositivi mobili

## Link Esterni

Il sito include un link esterno al sistema di prenotazione del Tecnam P96:
https://fringuello79.github.io/prenotazione-p96/

## Visualizzazione Locale

Per visualizzare il sito localmente, aprire il file `index.html` in un browser web o utilizzare un server HTTP locale:

```bash
python3 -m http.server 8080
```

Poi navigare a `http://localhost:8080`
